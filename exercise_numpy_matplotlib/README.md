# Opis

Task 2: Kod generuje i wyświetla krzywą Lissajous na podstawie parametrów amplitud A i B, częstotliwości a i b, oraz przesunięcia fazowego δ, które są przekazywane jako argumenty wiersza poleceń.<br />
